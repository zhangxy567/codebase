如何使用sublime Text、JDK写java文件及编译

sublime相当于一个大型记事本，JDK用来编译与运行

1、下载sublime text、JDK

2、在电脑中新建文本文档，打开方式选择sublime、开始写代码

注意，文件名需要和类名一致，比如

class Test{

}

public class helloworld{

   public static void main(String[] args){
      
System.out.println("hello");
   }
注意类名和文件名大小写也要一致，不然就后面编译不成功

}

3、打开CMD，一般路径为

 如果代码存在C盘，就直接在光标输入“cd ”加1中文件路径复制过去，回车

如果在其他盘，首先输入比如D盘“D：”回车，再重复上述步骤

4、然后变成字节码文件：输入“Javac 文件名.java"回车

5、输入”java 文件名“回车，即可编译成功
