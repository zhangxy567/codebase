4.Sublime Text 4 安装工具包

常用的插件

  1. ChineseLocalizations —— 中文汉化包
  2. Emmet（原名 Zen Coding）——快速编写html/css的方法
  3. ConvertToUTF-8 ——解决乱码问题
  4. Alignment——代码对齐插件
  5. AutoPrefixer ——css3前缀自动添加插件
  6. JS Fromat —— 格式化JS
  7. Minifier——JS压缩插件
  8. Sublime CodeIntel——代码自动提示
  9. SideBarEnhancements——侧边栏插件
  10. Keymaps ——快捷键查找等
  11. CSScomb——CSS属性排序

汉化工具包安装

点击菜单 【Preferences】 -> 【Package Control】-> 选择【Install Package】;出现一个新的弹窗。

再输入“【ChineseLocalizations 】。

安装完成之后，温馨提示：可自由切换，到此为止，可以看出中文已经安装成功。

随后可以在【帮助】【Language】中切换自己需要的语言。

html文件的快捷编写功能（在html中输入【！】有接口生成的快捷方式）

点击菜单 【Preferences】 -> 【Package Control】-> 选择【Install Package】;出现一个新的弹窗。

再输入“【Emmet 】。安装需要时间，耐心等待下，直到弹出对应的文件。

安装完成之后，就可以在html中输入！，出现html的基本代码啦。

5.删除工具包

在软件打开状态下，操作ctrl+shift+p
输入remove package

选中想要删除的工具包删除即可
