解决方案

  • win+R 输入regedit.exe 打开注册表

  • 按照层次找到：HKEY_LOCAL_MACHINE–>SOFTWARE–>Microsoft–>Windows–>CurrentVersion–>Explorer–>ShellIconOverlayIdentifiers

  • 将Tortoise开头的几个文件夹重命名，在前面敲几个空格，使得这几个文件夹名称的前面空格最多。

  • 最后重启一下任务管理器（快捷键：ctrl+shift+esc），就可以了
